### Initial Assumptions:
- **Goal:** Create a playable, browser-based clone of the classic game "Snake".
- **Key Stakeholders/Audience:** The user.
- **Context/Environment:** A web-based game using HTML, CSS, and vanilla JavaScript. It can be run by opening a single HTML file in a browser.
- **Core Requirements/Deliverables:**
    - A controllable snake on a grid.
    - The snake grows when it eats food.
    - The game ends on collision with a wall or the snake's own body.
    - A visible score display.
    - Arrow key controls for the snake.
