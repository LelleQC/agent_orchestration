# Doodle Jump Ultimate

This is a visually enhanced version of the classic Doodle Jump game.

## Features

*   Custom player character
*   Custom platforms
*   Parallax scrolling background
*   Particle effects
*   Sound effects
*   Start and game-over screens

## How to Play

1.  Open the `index.html` file in your web browser.
2.  Press `Enter` to start the game.
3.  Use the `ArrowLeft` and `ArrowRight` keys to move the character.
4.  The goal is to climb as high as possible by jumping on the platforms.
