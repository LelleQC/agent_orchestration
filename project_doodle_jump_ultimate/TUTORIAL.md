# Doodle Jump Ultimate Tutorial

This tutorial will guide you through playing Doodle Jump Ultimate.

## Step 1: Open the Game

Open the `index.html` file in your favorite web browser. You will see the start screen with the game title.

## Step 2: Start the Game

Press the `Enter` key to start the game. The character will start jumping automatically.

## Step 3: Control the Character

Use the `ArrowLeft` and `ArrowRight` keys on your keyboard to move the character left and right. The goal is to land on the platforms to keep climbing.

## Step 4: Game Over

If you miss a platform and fall to the bottom of the screen, the game is over. You will see the game-over screen. Press `Enter` to play again.

Enjoy the game!
