### Initial Assumptions:
- **Goal:** Create a playable, browser-based clone of the classic game "Breakout".
- **Key Stakeholders/Audience:** The user.
- **Context/Environment:** A web-based game using HTML, CSS, and vanilla JavaScript.
- **Core Requirements/Deliverables:**
    - A paddle at the bottom of the screen, controlled by the user's mouse.
    - A ball that moves and bounces off the screen edges and the paddle.
    - A grid of bricks at the top of the screen.
    - Bricks are destroyed when hit by the ball.
    - The game is lost if the ball falls below the paddle.
    - The game is won when all bricks are destroyed.

### Primary Phases:
-   **Phase 1: Scaffolding & Basic Setup:** Create the `index.html`, `style.css`, and `script.js` files.
-   **Phase 2: Game Element Implementation:** Define and draw the paddle, ball, and bricks.
