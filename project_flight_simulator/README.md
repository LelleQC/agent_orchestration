# Simple Flight Simulator

This project is a simple, browser-based 3D flight simulator created as a demonstration of an autonomous AI agent's capabilities.

## Technology

*   **3D Rendering:** [Three.js](https://threejs.org/)
*   **Web Server:** [serve](https://www.npmjs.com/package/serve)

## Features

*   **Basic 3D Scene:** A ground plane and a sky-colored background.
*   **Simple Aircraft:** A basic 3D model of an aircraft.
*   **Third-Person Camera:** The camera follows the aircraft.
*   **Keyboard Controls:** Use the arrow keys to control the aircraft's pitch and roll.
*   **Forward Movement:** The aircraft moves forward in the direction it is pointing.

## How to Run

1.  Install the dependencies:
    ```bash
    npm install
    ```
2.  Start the web server:
    ```bash
    npm start
    ```
3.  Open your browser and navigate to the URL provided by the web server (usually `http://localhost:3000`).
